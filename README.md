## Fonctionnement

Avec ce système, il n'est plus nécessaire de lancer sa machine virtuel REDS à chaque fois que l'on veut utiliser Qt pour PCO. Qt se lance dans un conteneur Docker avec toutes les dépendences nécessaire pour travailler le cours de PCO 2020.
